Semantic Alias Tokens

Components always reference these — not raw palette values. One change here rethemes the app.
--color-primary
CTA buttons, primary actions
--color-primary-hover
Button hover state
--color-primary-active
Pressed / active state
--color-primary-subtle
Hover bg, selected rows
--color-primary-tint
Tag / badge backgrounds
--color-primary-border
Focus ring, active borders
--color-bg
Page background
--color-surface
Card / panel background
--color-border
Default dividers, lines
--color-text
Primary body copy
--color-text-secondary
Captions, metadata
--color-text-tertiary
Placeholders, hints
--color-danger
Error, destructive actions
--color-danger-subtle
Error banner background
--color-danger-text
Error text on light bg (accessible)
--color-warning
Alert, degraded state
--color-warning-subtle
Warning banner background
--color-warning-text
Warning text on light bg
--color-success
Confirmation, positive
--color-success-subtle
Success banner background
--color-success-text
Success text on light bg
